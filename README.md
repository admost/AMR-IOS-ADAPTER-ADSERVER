# AMRAdapterAdserver

Adserver adapter for AMRSDK. Allows publishers to mediate Adserver banner, native, interstitial and video ads in AMRSDK.

## Requirements

- iOS 13.0+
- Xcode 15+

## Installation

### Swift Package Manager

In Xcode: **File → Add Package Dependencies** and enter the repository URL:

```
https://github.com/admost/AMR-IOS-ADAPTER-ADSERVER.git
```

### CocoaPods

Add the following to your `Podfile`:

```ruby
pod 'AMRAdapterAdserver'
```

Then run:

```sh
pod install
```
